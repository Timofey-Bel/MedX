## 🎨 Архитектура

### Модульная структура
- Каждая секция страницы вынесена в отдельный компонент
- CSS организован по компонентам (один файл = один компонент)
- JavaScript функционал изолирован в отдельном файле

### Следование стандартам Laravel
- Использование Blade шаблонизатора
- Правильная структура директорий
- Использование `@push`/`@stack` для скриптов и стилей
- Blade директивы на отдельных строках

### Адаптивный дизайн
- Desktop-first подход
- Media queries для планшетов и мобильных
- Flexbox и Grid для layout

## 🚀 Доступные страницы

После запуска проекта доступны:

1. **Главная страница MedX**
   - URL: `http://localhost:8000/medx`
   - Роут: `medx.showcase`

2. **Страница входа**
   - URL: `http://localhost:8000/medx/login`
   - Роут: `medx.login`

3. **Страница регистрации**
   - URL: `http://localhost:8000/medx/register`
   - Роут: `medx.register`

## 🔧 Как запустить

```bash
# 1. Установить зависимости
composer install

# 2. Настроить окружение
cp .env.example .env
php artisan key:generate

# 3. Очистить кеши
php artisan optimize:clear

# 4. Запустить сервер
php artisan serve

# 5. Открыть в браузере
http://localhost:8000/medx
```
