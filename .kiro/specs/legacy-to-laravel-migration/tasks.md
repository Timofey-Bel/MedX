## Примечания

- Задачи с постфиксом `*` являются опциональными и могут быть пропущены для ускорения разработки
- Каждая задача ссылается на конкретные требования для отслеживаемости
- Контрольные точки обеспечивают инкрементальную валидацию
- Все SQL-запросы должны использовать DB::select() или DB::selectOne()
- Все комментарии в коде должны быть на русском языке
- Knockout.js ViewModels должны следовать единому стандарту из legacy системы
- Тесты должны использовать транзакции для безопасной работы с рабочей БД
