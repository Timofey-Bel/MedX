# Перетаскиваемые иконки на рабочем столе - Windows 10 Style

## Дата реализации
2026-03-02

## Описание
Реализован функционал перетаскивания иконок на рабочем столе администратора в стиле Windows 10 с сохранением позиций в localStorage.

## Изменения в файле `resources/views/admin/desktop/index.blade.php`

### 1. Обновлены CSS стили для `.shortcut`

**Добавлено:**
- `position: absolute` - для возможности свободного позиционирования
- Улучшен hover эффект: `rgba(255, 255, 255, 0.15)` + border `rgba(255, 255, 255, 0.3)`

**Новые классы:**
```css
.shortcut.selected {
    background: rgba(255, 255, 255, 0.25);
    border: 2px solid rgba(255, 255, 255, 0.5);
}

.shortcut.dragging {
    opacity: 0.5;
    cursor: move !important;
}
```

### 2. Обновлен контейнер `.desktop-shortcuts`

**Было:**
```css
.desktop-shortcuts {
    position: absolute;
    top: 20px;
    left: 20px;
    display: flex;
    flex-direction: column;
    gap: 20px;
}
```

**Стало:**
```css
.desktop-shortcuts {
    position: absolute;
    top: 20px;
    left: 20px;
}
```

Убраны `display: flex`, `flex-direction: column`, `gap: 20px` для возможности свободного позиционирования иконок.

### 3. Добавлен JavaScript для Drag & Drop

**Функционал:**
- Перетаскивание иконок мышью
- Плавное перемещение без привязки к сетке
- Ограничение перемещения границами рабочего стола
- Сохранение позиций в localStorage
- Выделение иконки при клике (класс `.selected`)
- Визуальная обратная связь при перетаскивании (класс `.dragging`)

**Ключевые особенности:**

1. **Сохранение позиций:**
   - Ключ: `shortcut-pos-shortcut-{index}`
   - Формат: `{"left": 100, "top": 200}`
   - При первой загрузке иконки располагаются в центре рабочего стола

2. **Начальное позиционирование:**
   - Иконки размещаются в центре рабочего стола при первой загрузке
   - Позиция вычисляется как: `(desktopRect.width / 2) - 50` и `(desktopRect.height / 2) - 50`
   - Смещение 50px учитывает половину размера иконки для центрирования

3. **Ограничение границ:**
   - Иконки не могут выйти за пределы рабочего стола
   - Учитывается размер иконки (100px)

4. **Выделение:**
   - При клике на иконку она выделяется (класс `.selected`)
   - Предыдущее выделение снимается

## Использование

### Перетаскивание иконки
1. Наведите курсор на иконку
2. Зажмите левую кнопку мыши
3. Перетащите иконку в нужное место
4. Отпустите кнопку мыши
5. Иконка сохранит позицию в том месте, где вы её отпустили

### Сброс позиций
Для сброса всех позиций иконок выполните в консоли браузера:
```javascript
// Удалить все сохраненные позиции
for (var i = 0; i < localStorage.length; i++) {
    var key = localStorage.key(i);
    if (key.startsWith('shortcut-pos-')) {
        localStorage.removeItem(key);
    }
}
location.reload();
```

## Технические детали

### Структура кода

```javascript
(function() {
    var shortcuts = document.querySelectorAll('.shortcut');
    var draggedElement = null;
    var offsetX = 0;
    var offsetY = 0;
    
    shortcuts.forEach(function(shortcut, index) {
        // 1. Загрузка сохраненной позиции
        // 2. Установка начальной позиции (центр рабочего стола)
        // 3. Обработчики событий:
        //    - mousedown (выделение)
        //    - dragstart (начало перетаскивания)
        //    - dragend (конец перетаскивания)
    });
    
    // Разрешение drop на рабочем столе
    desktop.addEventListener('dragover', ...);
    desktop.addEventListener('drop', ...);
})();
```

### События

1. **mousedown** - выделение иконки
   - Снимает класс `.selected` со всех иконок
   - Добавляет класс `.selected` к текущей иконке

2. **dragstart** - начало перетаскивания
   - Сохраняет ссылку на перетаскиваемый элемент
   - Вычисляет смещение курсора относительно иконки
   - Добавляет класс `.dragging`

3. **dragend** - конец перетаскивания
   - Вычисляет новую позицию
   - Ограничивает границами рабочего стола
   - Сохраняет позицию в localStorage
   - Убирает класс `.dragging`

### Вычисление позиции

```javascript
// Позиция относительно рабочего стола
var newLeft = e.clientX - desktopRect.left - offsetX;
var newTop = e.clientY - desktopRect.top - offsetY;

// Ограничение границами
newLeft = Math.max(0, Math.min(newLeft, desktopRect.width - 100));
newTop = Math.max(0, Math.min(newTop, desktopRect.height - 100));
```
