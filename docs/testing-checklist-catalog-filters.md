# Чек-лист тестирования фильтров каталога

## Перед тестированием

1. Убедиться, что сервер запущен: `php artisan serve` или через локальный веб-сервер
2. Открыть браузер с инструментами разработчика (F12)
3. Очистить кэш браузера (Ctrl+Shift+Del)

## Базовая функциональность

### 1. Открытие каталога
- [ ] Открыть http://sfera/catalog
- [ ] Страница загружается без ошибок
- [ ] Отображаются товары
- [ ] Отображаются фильтры в боковой панели

### 2. Фильтр по авторам
- [ ] Выбрать одного автора
- [ ] Нажать "Применить фильтр"
- [ ] Страница перезагружается
- [ ] Отображаются только товары выбранного автора
- [ ] Чекбокс автора остается отмеченным
- [ ] URL содержит параметр `?author[]=...`

### 3. Фильтр по возрасту
- [ ] Выбрать один возраст
- [ ] Нажать "Применить фильтр"
- [ ] Отображаются только товары для выбранного возраста
- [ ] Чекбокс возраста остается отмеченным
- [ ] URL содержит параметр `?age[]=...`

### 4. Фильтр по сериям
- [ ] Выбрать одну серию
- [ ] Нажать "Применить фильтр"
- [ ] Отображаются только товары выбранной серии
- [ ] Чекбокс серии остается отмеченным
- [ ] URL содержит параметр `?seriya[]=...`

### 5. Фильтр по типам товаров
- [ ] Выбрать один тип товара
- [ ] Нажать "Применить фильтр"
- [ ] Отображаются только товары выбранного типа
- [ ] Чекбокс типа остается отмеченным
- [ ] URL содержит параметр `?product_type[]=...`

### 6. Фильтр по тематикам

- [ ] Нажать "Применить фильтр"
- [ ] Отображаются только товары выбранной тематики
- [ ] Чекбокс тематики остается отмеченным
- [ ] URL содержит параметр `?topic[]=...`

## Комбинированные фильтры

### 7. Несколько фильтров одновременно
- [ ] Выбрать автора + возраст
- [ ] Нажать "Применить фильтр"
- [ ] Отображаются товары, соответствующие ОБОИМ условиям
- [ ] Оба чекбокса остаются отмеченными
- [ ] URL содержит оба параметра

### 8. Все фильтры одновременно
- [ ] Выбрать по одному значению из каждого фильтра
- [ ] Нажать "Применить фильтр"
- [ ] Отображаются товары, соответствующие ВСЕМ условиям
- [ ] Все чекбоксы остаются отмеченными
- [ ] URL содержит все параметры

### 9. Несколько значений одного фильтра
- [ ] Выбрать 2-3 автора
- [ ] Нажать "Применить фильтр"
- [ ] Отображаются товары ЛЮБОГО из выбранных авторов
- [ ] Все выбранные чекбоксы остаются отмеченными

## Пагинация с фильтрами

### 10. Пагинация сохраняет фильтры
- [ ] Выбрать фильтр (например, автора)
- [ ] Нажать "Применить фильтр"
- [ ] Перейти на страницу 2
- [ ] Фильтр остается активным
- [ ] URL содержит параметры фильтра И пагинации: `?author[]=...&page=2`
- [ ] Отображаются товары со страницы 2, соответствующие фильтру

### 11. Навигация между страницами
- [ ] Выбрать фильтр
- [ ] Перейти на страницу 2
- [ ] Вернуться на страницу 1
- [ ] Фильтр остается активным на всех страницах

## Известные ограничения (документированы)

- Фильтр по возрасту работает по точному совпадению (не по диапазону)
- Максимум 4 автора отображается в фильтре (можно изменить в FilterService)
- Фильтры применяются через AND (все условия должны выполняться)
- Нет AJAX-обновления (будет в следующих версиях)
- Нет счетчиков товаров для каждого значения фильтра (будет в следующих версиях)

## Критические ошибки (требуют немедленного исправления)

- [ ] 500 Internal Server Error при применении фильтра
- [ ] SQL-ошибки в логе
- [ ] Фильтры не сохраняются при пагинации
- [ ] Неверные результаты фильтрации
- [ ] Дублирование товаров в результатах
