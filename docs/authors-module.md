# Модуль "Авторы" (Authors)

## Обзор

Модуль авторов позволяет пользователям просматривать список всех авторов и книги конкретного автора.

**Миграция из legacy:** `site/modules/sfera/authors/` и `site/modules/sfera/author/`

**Дата создания:** 2026-02-27

**Статус:** ✅ Реализовано

---

## Архитектура

### Backend (Laravel)

#### AuthorController (`app/Http/Controllers/AuthorController.php`)

Контроллер для работы со страницами авторов.

**Методы:**

1. `index()` - Список всех авторов
   - Получает всех авторов с количеством книг из таблицы `authors`
   - Группирует авторов по первой букве для удобного отображения
   - Сортирует по алфавиту
   - Route: `GET /authors`

2. `show(Request $request)` - Страница автора с его книгами
   - Получает имя автора из параметра `author_name`
   - Проверяет существование автора в БД
   - Получает все товары автора с ценами, изображениями и рейтингами
   - Реализует пагинацию (20 товаров на страницу)
   - Route: `GET /author?author_name={name}`

**Особенности:**
- Использует прямые SQL-запросы через `DB::select()`, `DB::selectOne()`
- Интегрируется с ProductService для получения рейтингов
- Все комментарии на русском языке

## Frontend

### Blade Templates

#### Список авторов (`resources/views/authors/index.blade.php`)

Страница со списком всех авторов, сгруппированных по алфавиту.

**Особенности:**
- Расширяет `layouts.app` для единообразия с остальными страницами
- Группировка по первой букве имени
- Отображение количества книг для каждого автора
- Breadcrumbs навигация
- Конвертировано из legacy `authors.tpl`

#### Страница автора (`resources/views/authors/show.blade.php`)

Страница с книгами конкретного автора.

**Особенности:**
- Расширяет `layouts.app` для единообразия с остальными страницами

- Конвертировано из legacy `author.tpl`

## База данных

### Таблица `authors`

```sql
CREATE TABLE authors (
    id INT AUTO_INCREMENT PRIMARY KEY,
    product_id VARCHAR(50) NOT NULL,
    author_name VARCHAR(255) NOT NULL,
    INDEX idx_product_id (product_id),
    INDEX idx_author_name (author_name)
);
```

**Поля:**
- `id` - первичный ключ
- `product_id` - ID товара (связь с таблицей products)
- `author_name` - имя автора

**Примечание:** Один товар может иметь несколько авторов (связь many-to-many через эту таблицу).
