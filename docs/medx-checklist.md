## 🚀 Быстрый запуск

```bash
# 1. Установить зависимости
composer install

# 2. Настроить окружение
cp .env.example .env
php artisan key:generate

# 3. Очистить кеши
php artisan optimize:clear

# 4. Запустить сервер
php artisan serve

# 5. Открыть в браузере
# http://localhost:8000/medx
```
