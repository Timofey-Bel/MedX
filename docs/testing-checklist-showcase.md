## Обзор

Этот документ содержит чек-лист для проверки функционала главной страницы интернет-магазина "Сфера".

## Контроллер: ShowcaseController

### Проверка методов контроллера

- [x] **getMainCarouselBanners()** - получение баннеров для главного карусели
  - Запрос к таблице `banners`
  - Фильтрация по `active = 1`
  - Сортировка по `sort ASC`
  - Обработка ошибок с логированием

- [x] **getProductCarouselData()** - получение товаров для карусели товаров
  - Запрос к таблицам `products` и `prices`
  - Фильтрация по `is_new = 1`
  - Случайная выборка (inRandomOrder)
  - Лимит 3 товара
  - Добавление изображений через ProductService::getProductImageUrl()

- [x] **getPromoCarouselData()** - получение данных для промо-карусели
  - Аналогично getProductCarouselData()
  - Случайная выборка новых товаров
  - Лимит 3 товара

- [x] **getPopularCategories()** - получение популярных категорий
  - Запрос к таблицам `popular_categories` и `tree`
  - Фильтрация по `active = 1`
  - Сортировка по `sort ASC`
  - Преобразование в массивы для Blade

- [x] **getTop10Products()** - получение топ-10 товаров
  - Запрос к таблицам `top10_products`, `products`, `prices`
  - Фильтрация по `active = 1`
  - Сортировка по `sort ASC`
  - Лимит 10 товаров
  - **ОБЯЗАТЕЛЬНО**: Добавление рейтинга через ProductService::getProductRating()
  - **ОБЯЗАТЕЛЬНО**: Добавление изображений через ProductService::getProductImageUrl()
  - Округление цены
  - Fallback для quantity (99)

- [x] **getProductReviews()** - получение отзывов о товарах
  - Запрос к таблице `product_reviews`
  - Фильтрация по `active = 1`
  - Сортировка по `sort ASC`
  - Лимит 10 отзывов
  - Преобразование в массивы для Blade

- [x] **getRandomProducts()** - получение случайных товаров (новинки)
  - Запрос к таблицам `products` и `prices`
  - Фильтрация по `is_new = 1`
  - Случайная выборка (inRandomOrder)
  - Лимит 12 товаров
  - **ОБЯЗАТЕЛЬНО**: Добавление рейтинга через ProductService::getProductRating()
  - **ОБЯЗАТЕЛЬНО**: Добавление изображений через ProductService::getProductImageUrl()
  - Округление цены
  - Fallback для quantity (99)
