# Тестирование галереи товара с Swiper.js

## Checkpoint 1: Базовая галерея (Task 5)

### Что проверяем

Базовая функциональность галереи изображений на странице товара.

### Подготовка

1. Убедитесь, что файлы Swiper.js загружены:
   - `public/assets/libs/swiper/swiper-bundle.min.css`
   - `public/assets/libs/swiper/swiper-bundle.min.js`

2. Откройте страницу товара с несколькими изображениями (10+):
   - Пример: http://sfera/product/00-0002266 (710 изображений)

### Тест-кейсы

#### 1. Вертикальная карусель миниатюр

**Ожидаемое поведение:**
- [ ] Миниатюры отображаются вертикально слева от основного изображения
- [ ] Видно 5 миниатюр одновременно
- [ ] Можно прокручивать миниатюры вверх/вниз (колесико мыши или свайп)
- [ ] Активная миниатюра выделена синей рамкой

**Как проверить:**
1. Откройте страницу товара
2. Посмотрите на левую часть галереи
3. Попробуйте прокрутить миниатюры колесиком мыши
4. Проверьте, что активная миниатюра имеет синюю рамку

#### 2. Клик по миниатюре меняет основное изображение

**Ожидаемое поведение:**
- [ ] При клике на миниатюру основное изображение меняется
- [ ] Переход плавный (анимация)
- [ ] Кликнутая миниатюра становится активной (синяя рамка)

**Как проверить:**
1. Кликните на любую миниатюру
2. Убедитесь, что основное изображение изменилось
3. Проверьте, что миниатюра выделена синей рамкой

#### 3. Навигационные кнопки основного Swiper

**Ожидаемое поведение:**
- [ ] Кнопки "вперед" и "назад" видны на основном изображении
- [ ] При клике на кнопку изображение меняется
- [ ] Миниатюра автоматически синхронизируется
- [ ] Кнопки отключаются на первом/последнем слайде

**Как проверить:**
1. Наведите курсор на основное изображение
2. Кликните на стрелку "вперед" (→)
3. Убедитесь, что изображение изменилось
4. Проверьте, что активная миниатюра тоже изменилась
5. Пролистайте до конца - кнопка "вперед" должна стать неактивной

#### 4. Клик по основному изображению открывает lightbox

**Ожидаемое поведение:**
- [ ] При клике на основное изображение открывается модальное окно
- [ ] Модальное окно занимает весь экран
- [ ] Фон затемнен (черный с прозрачностью)
- [ ] Изображение отображается в центре
- [ ] Видна кнопка закрытия (×) в правом верхнем углу

**Как проверить:**
1. Кликните на основное изображение
2. Убедитесь, что открылось модальное окно
3. Проверьте наличие кнопки закрытия

#### 5. Навигация в lightbox

**Ожидаемое поведение:**
- [ ] Стрелки навигации работают (← →)
- [ ] Можно листать клавишами клавиатуры (← →)
- [ ] Отображается счетчик слайдов (например, "3 / 10")
- [ ] Можно свайпать на touch-устройствах

**Как проверить:**
1. Откройте lightbox (клик по основному изображению)
2. Кликните на стрелку "вперед"
3. Нажмите клавишу ← на клавиатуре
4. Проверьте счетчик слайдов внизу

#### 6. Закрытие lightbox

**Ожидаемое поведение:**
- [ ] Кнопка × закрывает модальное окно
- [ ] Клавиша Escape закрывает модальное окно
- [ ] Клик по затемненному фону закрывает модальное окно
- [ ] После закрытия прокрутка страницы восстанавливается

**Как проверить:**
1. Откройте lightbox
2. Нажмите кнопку × - окно должно закрыться
3. Откройте снова, нажмите Escape - окно должно закрыться
4. Откройте снова, кликните по черному фону - окно должно закрыться

### Проверка в консоли браузера

Откройте DevTools (F12) и проверьте:

1. **Network (Сеть):**
   - [ ] `swiper-bundle.min.css` загружен (статус 200)
   - [ ] `swiper-bundle.min.js` загружен (статус 200)
   - [ ] `product-gallery.css` загружен (статус 200)
   - [ ] `product-gallery.js` загружен (статус 200)

2. **Console (Консоль):**
   - [ ] Нет ошибок JavaScript
   - [ ] Видно сообщение: "ProductGallery initialized successfully"
   - [ ] Можно проверить: `typeof Swiper` должен быть "function"
   - [ ] Можно проверить: `window.productGallery` должен быть объектом

### Известные проблемы

Если что-то не работает, проверьте:

1. **Swiper не загружается:**
   - Проверьте пути к файлам в `resources/views/product/show.blade.php`
   - Убедитесь, что используется `@push('scripts')`, а не `@section('scripts')`

2. **Миниатюры не вертикальные:**
   - Проверьте CSS в `product-gallery.css`
   - Убедитесь, что `direction: 'vertical'` в `product-gallery.js`

3. **Lightbox не открывается:**
   - Проверьте консоль на ошибки JavaScript
   - Убедитесь, что `product-gallery.js` загружен после `swiper-bundle.min.js`
