# Помодоро Таймер

## Реализованный функционал

- 25 минут работы → 5 минут короткий перерыв → после 4 кругов 15 минут длинный перерыв
- Редактирование времени до запуска (клик на минуты/секунды)
- Смена цветовой схемы при перерыве (break-mode)
- Модальное окно с информацией (клик на "?")
- Иконки: play, stop, return из `public/images/main/`
- **Глобальные уведомления** - всплывают справа снизу на любой странице
- **Сохранение состояния** - таймер продолжает работу при переходе между страницами
- **Фоновый режим** - таймер работает даже когда вы не на странице showcase

## Как работает

1. Запустите таймер на странице showcase
2. Перейдите на любую другую страницу - таймер продолжит работу
3. Когда круг закончится - появится уведомление с кнопкой запуска следующего этапа
4. Состояние сохраняется в localStorage и восстанавливается при перезагрузке

## Уведомления

При завершении рабочего периода:
- Заголовок: "Ура, отдыхать!"
- Кнопка: "ЗАПУСТИТЬ ТАЙМЕР ОТДЫХА"

При завершении перерыва:
- Заголовок: "Время работать!"
- Кнопка: "ЗАПУСТИТЬ ТАЙМЕР РАБОТЫ"

Уведомления автоматически закрываются через 10 секунд или по клику на крестик.

## Быстрый тест

1. Откройте `http://localhost/main_showcase`
2. Найдите секцию "Помодоро – таймер"
3. Кликните на минуты/секунды чтобы изменить время (например, 00:05)
4. Нажмите Play → таймер запустится
5. Перейдите на другую страницу → таймер продолжит работу
6. Дождитесь окончания → появится уведомление справа снизу

## Отладка

Откройте консоль (F12), должны быть сообщения:
```
Загрузка скрипта помодоро таймера
Инициализация помодоро таймера
Таймер успешно инициализирован
```

На страницах без UI таймера:
```
UI элементы не найдены, запуск в фоновом режиме
Запуск фонового режима таймера
```

## Файлы

- `public/js/pomodoro-timer.js` - основная логика + система уведомлений
- `public/js/pomodoro-debug.js` - отладка (можно удалить после теста)
- `public/test-timer.html` - тестовая страница
- `resources/views/main_showcase.blade.php` - HTML таймера
- `public/css/main_styles.css` - стили таймера + уведомлений
- `resources/views/layouts/app.blade.php` - подключение глобального скрипта

## Технические детали

- Состояние хранится в `localStorage` с ключом `pomodoroState`
- Таймер работает через `setInterval` с проверкой каждую секунду
- При переходе между страницами время корректируется с учетом прошедших секунд
- Уведомления создаются динамически через `document.createElement`
