# Календарь ежедневных входов

## Функционал
- Недельные слайды (7 дней в сетке) на главной странице
- **Полный календарь по месяцам** - открывается при клике на стрелку
- Автоматическая отметка текущего дня
- Визуальные индикаторы: ✓ посещен, ✗ пропущен, ○ нейтральный, ★ первый вход
- Навигация стрелками между неделями
- Сохранение в localStorage

## Модальное окно календаря
При клике на стрелку справа открывается полный календарь:
- Показывает один месяц за раз
- Название месяца отображается между кнопками навигации
- **Счетчик дней подряд** - показывает сколько дней пользователь заходит без пропусков (🔥 3 дня подряд)
- Кнопки "Назад" и "Вперед" для переключения между месяцами
- Заголовки дней недели (Пн, Вт, Ср, Чт, Пт, Сб, Вс)
- Первый вход отмечен звездочкой ★
- Посещенные дни - синий фон с галочкой ✓
- Пропущенные дни - крестик ✗
- Дни до регистрации - полупрозрачные
- Будущие дни - полупрозрачные

## Логика крестиков
Крестики появляются только для дней **после первого входа** пользователя. Дни до регистрации не помечаются.

Условия для крестика:
1. День в прошлом
2. День не был посещен
3. День >= дате первого входа

## Сохраняемые данные

```javascript
// Дата первого входа
localStorage: medx_first_visit_date
Формат: "2026-03-20T00:00:00.000Z"

// Посещенные дни
localStorage: medx_visited_days
Формат: ["2026-03-27", "2026-03-26"]
```

## Тестирование

### Быстрый тест
1. Откройте `http://localhost/test-calendar.html`
2. Проверьте отображение 7 дней в сетке
3. Кликните на стрелку справа - откроется полный календарь
4. Используйте кнопку "Установить первый вход неделю назад"

### Консольные команды
```javascript
// Посмотреть данные
localStorage.getItem('medx_visited_days')
localStorage.getItem('medx_first_visit_date')

// Очистить
localStorage.removeItem('medx_visited_days')
localStorage.removeItem('medx_first_visit_date')
location.reload()

// Симуляция входа неделю назад
const weekAgo = new Date()
weekAgo.setDate(weekAgo.getDate() - 7)
localStorage.setItem('medx_first_visit_date', weekAgo.toISOString())
location.reload()
```
